<h2>🧐 Features</h2>

Here're some of the project's best features:

*   On boarding screen
*   SharedPreferences
*   Dark and Light themes
*   Riverpod 2.5
*   Responsive design
*   clean architecture with MVVM pattern
*   Hive database & Hive adapter
*   Multiple gestures for ease of use
*   CRUD operations
*   Filter data for favorite tasks
*   Mark a task as done

<h2>💻 Built with</h2>

*   Flutter

*   Hive
